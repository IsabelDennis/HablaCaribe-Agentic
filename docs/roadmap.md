# Roadmap

## 21-Day Buildathon Goal

Build a focused, runnable demonstration of the adaptive learning engine behind HablaCaribe.

## Week 1: Foundation

- Confirm technical stack with mentors and collaborators
- Set up repository and development environment
- Build learner intake flow
- Implement ethics, safety, and suitability gate
- Create Purpose Agent
- Define dynamic learner profile schema
- Create first sample learner journeys

## Week 2: Adaptive Learning Loop

- Build Learning Designer Agent
- Build Learning Experience Engine
- Create a small learning content library
- Implement one or two learning experiences:
  - AI conversation
  - Writing activity
  - Simple game or challenge
- Build Assessment Agent
- Store learner progress and memory

## Week 3: Demo and Refinement

- Connect full adaptive loop
- Add speech-to-text and text-to-speech if feasible
- Test with sample learner scenarios
- Prepare runnable demo
- Document metrics, architecture, and next steps
- Prepare final demo video and README

## Five-Year Vision

Within five years, HablaCaribe aims to become a leading example of what Caribbean-centred, agentic learning can achieve.

Success will be measured by:

- operating across six Caribbean territories
- supporting 25,000+ learner accounts
- reaching at least 5,000 monthly active learners
- building partnerships with at least 75 schools, employers, universities, and regional organisations
- demonstrating measurable learner outcomes in communication, collaboration, and real-world goal attainment

Longer term, the adaptive learning engine behind HablaCaribe can become the foundation for a family of Caribbean-centred learning experiences developed through Learning to Live.
