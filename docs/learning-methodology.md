# Learning Methodology

HablaCaribe is built on a Caribbean-centred learning system developed through years of research, implementation, and testing across schools, online learning environments, and organisational development.

## Core belief

People do not learn only by receiving information.

They learn by:

- connecting learning to purpose
- building confidence
- practising in context
- receiving feedback
- reflecting on progress
- applying learning to real situations
- learning through relationships

## Purpose-first learning

HablaCaribe does not begin by asking:

> What language do you want to learn?

It begins by asking:

> What do you want to do?

The learner's purpose guides the pathway.

## Capability building

HablaCaribe is not only teaching language.

It is developing the capabilities needed to participate confidently across languages and cultures:

- communication
- cultural understanding
- confidence
- relationship-building
- real-world application
- reflective learning
- collaboration

## Why this matters

Translation tools help people understand a conversation.

HablaCaribe develops the capabilities needed to participate confidently in one.
