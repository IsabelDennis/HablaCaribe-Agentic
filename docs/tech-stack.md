# Anticipated Technical Stack

This stack reflects the current architecture and may be refined with technical collaborators, mentors, and Buildathon infrastructure.

## Learner Interface

- React or another suitable frontend framework

## Backend

- Python for backend orchestration and APIs

## Agent Orchestration

- OpenClaw or the agent orchestration framework recommended during the Buildathon

## AI Models

- Open-source large language model running on NVIDIA H200 compute
- Embedding models for retrieval and personalization
- Speech-to-text for learner voice input
- Text-to-speech for conversational learning

## Data and Memory

- PostgreSQL for learner profiles, progress, and memory
- Learning content library for activities, prompts, vocabulary, scenarios, and rubrics
- Object storage for media assets such as audio, images, worksheets, and activity files

## Development

- GitHub for version control
- H200 compute for agent workflows, model inference, speech processing, embeddings, and deployment support

## Design principle

The technology should serve the learning system. Agentic AI is the vehicle for delivering the Caribbean-centred learning methodology at scale.
