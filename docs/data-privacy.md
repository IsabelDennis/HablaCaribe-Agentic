# Data, Privacy, and Safeguarding

## Data involved

HablaCaribe may manage personal and educational data, including:

- learner account information
- learning goals
- progress records
- assessment results
- writing activities
- voice interactions or transcripts, where conversational features are enabled
- personalized learner profiles

The initial MVP does not anticipate collecting financial or healthcare data.

## Data protection approach

Planned safeguards include:

- authentication
- role-based access controls
- encrypted data transmission using HTTPS/TLS
- encrypted storage where appropriate
- separation of personally identifiable information from learning analytics where practical
- anonymized or aggregated reporting whenever individual identity is not required

## Minors and safeguarding

Because HablaCaribe may serve school-aged learners, additional safeguards will be built into the system design.

These include:

- parental consent where required
- human review for safety-sensitive goals or content
- educator review for high-stakes learning feedback
- learner control over goals and progression
- AI recommendations that support, not replace, the judgement of learners, parents, and educators

## External datasets

HablaCaribe may use public reference resources, open-source language and speech models, and educational reference datasets. It will also use proprietary learning content developed by Learning to Live and HablaCaribe.

External resources will be selected based on suitability and licensing terms. User-generated learning data will remain separate from reference datasets and will be used only to personalize each learner's pathway and improve the learning experience.
