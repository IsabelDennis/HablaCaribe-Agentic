# Contributing

HablaCaribe welcomes technical collaboration during the FutureCaribbean Buildathon.

## Current collaboration needs

We are especially interested in collaborators with experience in:

- AI engineering
- full-stack development
- agent orchestration
- speech-to-text and text-to-speech
- Python backend development
- React frontend development
- data privacy and responsible AI

## Guiding principle

The technology should serve the learning system.

HablaCaribe is not trying to replace teachers, learners, parents, or educators. The goal is to build an adaptive learning engine that supports better learning decisions while keeping humans in control.
